## Installation

```bash
$ npm install
```

## Configuration
Copy the files `config.example.ts` to `config.ts` and `datasources.example.ts` to `datasources.ts` and add appropriate values.

## Migration
Use the file in `./migrations` folder to run the initial db migration.

## Building the app

```bash
# watch build
$ npm run build:dev

# production build
$ npm run build
```

## Running the app

```bash
# debug + watch mode
$ npm run start:debug

# watch mode
$ npm run start:dev

# production mode
$ npm run start
```
